```
class he{
    public static void main(String[] args){
        int number = 1;	//初始值 1，以后再+2 递增上去
        int sum = 0;
        for ( ; number <100; number+=2 ){ 
            sum += number; 
        } 
        System.out.println("1+3+5+7+……+99= " +sum);
    }
}
```
